## Lambda function 
* A lambda function is a small anonymous function.
* A lambda function can take any number of arguments.
* It can only have __one expression__

1) Example of a basic Lambda Function:
```
 variable = lambda x:x**2
 print(variable(3))

Output :
9
```
2) Lambda function with more than one parameters
```
add = lambda x,y:x+y
print(add(3,5))
```
